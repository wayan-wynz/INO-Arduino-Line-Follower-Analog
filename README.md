# INO-Arduino-Line-Follower-Analog
Arduino Robot Line Follower menggunakan Mikorkontroler Arduino Uno. <br>
**Catu daya menggunakan  Baterai Lipo 2S (7,4v)**<br>
**Menggunakan 2 Buah SENSOR PHOTODIODA**

## Bahan-bahan Utama
- Arduino Uno
- Motor Driver L298N
- Sensor Photodioda
